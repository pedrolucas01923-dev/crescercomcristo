# **App Name**: FaithGrow Kids

## Core Features:

- Dynamic Urgency Header: Fixed header displaying a limited-time offer with a countdown timer.
- Engaging Hero Section: Visually appealing hero section showcasing the ebook collection with a clear call to action.
- Product Explanation Section: Detailed cards explaining what's included in the Christian Children's Collection, each accompanied by an icon and a miniature image.
- Differentiated Material Advantages: Short cards highlighting the collection's unique benefits, using icons and brief descriptions.
- Exclusive Bonus Display: Visually present available bonuses via cards with images and mini-CTAs to boost user engagement.
- Social Proof Integration: Showcase testimonials and reviews to build trust with potential users.
- Final Call to Action Block: Highlighted CTA block that visually summarizes and offers final call to action

## Style Guidelines:

- Primary color: Soft lavender (#E6E6FA) to evoke feelings of peace, reverence and learning.
- Background color: Light cream (#F5F5DC), which aligns with the peaceful mood and is only very lightly tinted from the primary color.
- Accent color: Pale Blue (#ADD8E6) that helps to draw users into call to actions and bonus buttons.
- Body and headline font: 'Alegreya' serif for an elegant and readable look.
- Use simple, modern icons with pastel fills to represent the values taught.
- Hero layout to be designed after the style of landing page from Yogi, keeping layout clean and easy to digest.
- Use a subtle pulsating animation on the header timer.
